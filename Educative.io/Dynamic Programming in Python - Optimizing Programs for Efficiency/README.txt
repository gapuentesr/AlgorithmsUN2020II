"The course promotes the change of perspective from recursive programming towards dynamic programming"

In the first chapter, i study the concept of recursion and also that of dynamic programming, as well as some applications.

In the second chapter the concept of Top-down dynamic programming using memorization is developed.

In the third chapter the concept of Bottom-Up dynamic programming using tabulation is studied.

Finally, some exercises are developed where various ways of solving problems are shown using the topics studied in the course.
